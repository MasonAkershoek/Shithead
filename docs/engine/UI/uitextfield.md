# UITextField
---
**UITextField** allows for geting typed user input in the game
### Usage
```UITextField.new(x,y,w,h,fontSize,args)```

### Parameters
| Parameter Name | Definition |
| ----------- | ----------- |
| x | X position to spawn at |
| y | Y position to spawn at |
| w | Width of the UITextField |
| h | Hight of the UITextField |
| fontSize | Size of the text displayed in the UITextField |
| args | A table containing optinal configurations |



### Args
| Argument Name | Default Value | Definition |
| ----------- | ----------- | ----------- |
| fontSize | 20 | Sets the font size for the text to be rendered with |
| maxLen | 10 | Sets the maximum about of characters that can be typed into the field |
| color | "WHITE" | Sets the background color of the UITextField |
| textColor | "BLACK" | Sets the text color of the UITextField |
| borderSize | 0 | Sets the size of the UITextField border |
| borderColor | "WHITE" | Sets the color of the UITextField Border |
| borderColorSelected | "WHITE" | Sets the color of the UITextField when selected |
| tempText | "" | Sets the text to be displayed in the UITextField before any input is entered |
| showLabel | false | Sets wether or not to show a label for the UITextField |
