# UITextField
---
**UITextField** allows for geting typed user input in the game
### Usage
```UITextField.new(x,y,w,h,fontSize,args)```

### Parameters
| Parameter Name | Definition |
| ----------- | ----------- |
| x | X position to spawn at |
| y | Y position to spawn at |
| w | Width of the UITextField |
| h | Hight of the UITextField |
| fontSize | Size of the text displayed in the UITextField |
| args | A table containing optinal configurations |



### Args
| Argument Name | Default Value | Definition |
| ----------- | ----------- | ----------- |
| fontSize | 20 | Sets the font size for the text to be rendered with |
| maxLen | 10 | Sets the maximum about of characters that can be typed into the field |
| color | "WHITE" | Sets the background color of the UITextField |
| textColor | "BLACK" | Sets the text color of the UITextField |
| borderSize | 0 | Sets the size of the UITextField border |
| borderColor | "WHITE" | Sets the color of the UITextField Border |
| borderColorSelected | "WHITE" | Sets the color of the UITextField when selected |
| tempText | "" | Sets the text to be displayed in the UITextField before any input is entered |
| showLabel | false | Sets wether or not to show a label for the UITextField |
| labelPos | left | Places the lable at one of 4 positions {"left", "top", "right", "bottom"} |
| labelAlignment | center | Aligns the label to the far left, far right or center of the UIText field when the labelPos is "top" or "bottom" {"left", "center", "right"}







### Variables
| Variable Name | Definition |
| ----------- | ----------- |
| T | Class identity |
| fontSize | Holds the fond side of the text displayed in the UITextField |
| selected | Tells wether the UITextField is currently selected and able to have input entered |
| maxLen | Holds the number of characters the UITextField can hold |
| color | Holds the color of the UITextField |
| textColor | Holds the color of the text diplayed in the UITextField |
| text | Holds the current input of the user into the UITextField |
| textGraphics | Holds the rendered text generated by the users input |
| borderSize | Holds the size of the UITextField border |
| borderColor | Holds the color of the UITextField border when unselected |
| borderColorSelected | Holds the color of the UITextField border when selected |
| tmpText | Holds the temp text that is displayed when the user hasnt input anything into the UITextField |
| tmpTextGraphics | Holds the renderd text of the tmpText |
| showLabel | Tells wether the UITextField label should be shown or not |
| labelPos | Holds the position around the UITextField that the label should be placed |
| labelAlignment | Tells if the UITextField label should be placed to the left right or center of the top or bottom |
| labelGraphics | Holds the renderd text of the UITextField label |



### Functions
| Function Name | Parameters | Definition |
| ----------- | ----------- | ----------- |
| UITextField.new | x, y, w, h, fontSize, args | Object constructor |
| UITextField:select | none | Handles selecting and unselecting the UITextField |
| UITextField:getText | none | Returns the current input in the UITextField |
| UITextField:addText | newChar | Adds a new character to the UITextField |
| UITextField:backSpace | none | Removes a character from the UITextField |
| UITextField:showCursor | none | Draws a flashing cursor in the UITextField |
| UITextField:update | dt (Delta time) | Updates the UITextField and its children |
| UITextField:draw | none | Draws the UITextField and its children |
